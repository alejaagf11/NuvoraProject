# Nuvora

## Descripción
Este proyecto es una aplicación para la gestión de finanzas personales que permite a los usuarios registrar ingresos, gastos, categorías y presupuestos.
El objetivo es ayudar a las personas a controlar su dinero y tomar mejores decisiones financieras.

## Funcionalidades
- Registro de usuarios
- Inicio de sesión
- Registro de ingresos y gastos
- Organización por categorías
- Creación de presupuestos
- Visualización de reportes financieros

## Tecnologías utilizadas
- Java
- Spring Boot
- MySQL
- Flutter
- Angular
- MySQL Workbench

## Base de datos
La base de datos se gestiona con MySQL y contiene las siguientes entidades principales:
- Usuario
- Transacción
- Categoría
- Presupuesto
- Tips
- MetasAhorro
